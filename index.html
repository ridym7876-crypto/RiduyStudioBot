🔐 Authentication কাস্টম API key সিস্টেম — নিরাপদ ও সহজ 

# API Key Format
Header: ABDUL_KHALEK_RIDUY_API-KEY
Format: Riduy + [30-35 random chars] + Apik
Length: 40-45 characters
Example: RiduyUZHAWGawyDxrPXwT66LLNwvneQxUXLEUcflApik

# Key একটা app-এ lock থাকে
bind_app: "my_app_name"  # optional — এক key, এক জায়গা

🚀 Quick Start ৩ স্টেপে শুরু করুন

1

API Key তৈরি করুন

generate_key action দিয়ে নতুন key তৈরি করুন।

2

Header যোগ করুন

প্রতিটা request-এ ABDUL_KHALEK_RIDUY_API-KEY header পাঠান।

3

AI কল করুন

যেকোনো endpoint call করুন — ফ্রি!

# 1. Key Generate
POST /riduyApi
{
  "action": "generate_key",
  "bind_app": "my_website"
}

# 2. Image তৈরি
POST /riduyApi
Header: ABDUL_KHALEK_RIDUY_API-KEY: Riduy...Apik
{
  "action": "text_to_image",
  "prompt": "a beautiful sunset over the ocean"
}

# 3. Code তৈরি
POST /riduyApi
Header: ABDUL_KHALEK_RIDUY_API-KEY: Riduy...Apik
{
  "action": "text_to_code",
  "prompt": "write a Python fibonacci function"
}
